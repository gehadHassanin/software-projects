Calculate a percentage portion of an amount, with rounding options.

Parameters
amountThe original amount.
rateThe percentage rate (as a decimal, e.g., 0.25 for 25%).
roundingRounding mode:
  • "HalfUp" → Round to nearest integer, ties away from zero.
  • "HalfEven" → Round to nearest even integer (banker's rounding).
Returns
The portion of the amount after applying the rate and rounding, or -1 if an invalid rounding mode is provided.
